# dotfiles/screen

Configuration for GNU [screen](https://www.gnu.org/software/screen/). `screenrc` configures the hardstatus line, sets the escape key, enables altscreen, uses 256-color terminfo, and bumps scrollback to 10,000 lines; it is symlinked to `~/.screenrc`.
